# Nextra Docs Starter 

This starter template includes the following features:

- [x] **Nextra 4**
- [x] **Search with Pagefind**

## Local Development

### Clone this repository

Using the GitHub CLI:
```bash
gh repo clone url
```

### Install
```bash
pnpm i
```

### Run the development server
```bash
pnpm dev
```
